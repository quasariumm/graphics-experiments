# DX Meshlets
This demo is my first experience with clustered geometry rendering (meshlets).

## Requirements
- A GPU with DX12 feature level 12.2 (DX Ultimate)

## Controls
WASD - Moves the camera  
E & Q - Moves the camera up and down  
Right Click + Mouse move - Pans the camera

Number 1 - Shaded rendering view  
Number 2 - Meshlet index debug view
